### Introduction
---
"NTU Vote" is an online voting system in National Taiwan University. This is the frontend React project for NTU Vote. 

### Prerequisites
---
    npm=^8 should be installed
    nodejs=^16 should be installed

### Usage
---
clone this repository /
cd to the directory
edit .env
```bash
npm install
npm start
```
